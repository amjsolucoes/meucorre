# Workflow: Criar Nova Tela

1. Criar o arquivo em app/(tabs)/nome-da-tela.tsx
2. Usar NativeWind para estilização
3. Buscar dados APENAS via hook em /hooks
4. Formulários com React Hook Form + Zod
5. Botões grandes com texto simples em português
6. Tratar loading e erro com mensagem amigável
7. Máximo 3 ações por tela
