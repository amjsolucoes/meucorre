## 💡 Dicas de ASO (App Store Optimization)

### Para App Store (iOS)
1. Use todas as 100 caracteres de palavras-chave
2. Não repita palavras do título nas keywords
3. Use vírgulas sem espaços: "palavra1,palavra2,palavra3"
4. Atualize keywords baseado em performance

### Para Google Play (Android)
1. Use palavras-chave naturalmente na descrição
2. Repita termos importantes 3-5 vezes
3. Use formatação (✓, •, emojis) para destacar
4. Primeira linha é crucial para conversão

### Geral
1. Screenshots com texto explicativo convertem mais
2. Vídeo preview aumenta downloads em 20-30%
3. Responda TODAS as avaliações (positivas e negativas)
4. Atualize o app regularmente (melhora ranking)
5. Incentive avaliações de usuários satisfeitos
